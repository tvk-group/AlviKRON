## Usage

- **Token explorers:** upload `icon-32.svg` or `logo-full.png` when updating token info on BaseScan.
- **Website / apps:** prefer `icon.svg` for UI; `logo-full.svg` or `logo-full.png` for marketing headers.
- **Do not** stretch, recolor, or alter the lightning-bolt **K** mark without approval.
