# Agent Skills (Cursor)

Design and UI skills installed for this project via [`npx skills add`](https://github.com/vercel-labs/agent-skills).

## Installed sources

| Source | Primary skills |
|--------|----------------|
| [Leonxlnx/taste-skill](https://github.com/Leonxlnx/taste-skill) | `design-taste-frontend`, `redesign-existing-projects`, `image-to-code`, … |
| [anthropics/skills](https://github.com/anthropics/skills/tree/main/skills/frontend-design) | `frontend-design` |
| [kylezantos/design-motion-principles](https://github.com/kylezantos/design-motion-principles) | `design-motion-principles` |
| [pbakaus/impeccable](https://github.com/pbakaus/impeccable) | `impeccable` |
| [nextlevelbuilder/ui-ux-pro-max-skill](https://github.com/nextlevelbuilder/ui-ux-pro-max-skill) | `ui-ux-pro-max`, `ckm-*` |

## Update

```bash
npx skills add https://github.com/Leonxlnx/taste-skill -y
npx skills add https://github.com/anthropics/skills --skill frontend-design -y
npx skills add https://github.com/kylezantos/design-motion-principles -y
npx skills add https://github.com/pbakaus/impeccable -y
npx skills add https://github.com/nextlevelbuilder/ui-ux-pro-max-skill -y
```

Lock file: `skills-lock.json` at repo root.
